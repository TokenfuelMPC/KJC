# KJC — KJ Christensen

Single-page marketing site for KJ Christensen ("The Quest, After Success").

- `index.html` — the complete static site (self-contained: inline CSS, Google Fonts via CDN).

## Preview locally
Open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000

## Deploy (GitHub Pages)
Settings → Pages → Deploy from branch `main`, root. Live at the Pages URL once enabled.
